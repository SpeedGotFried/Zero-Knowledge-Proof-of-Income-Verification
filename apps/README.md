# Frontend Portals (Templates)

This directory contains the scaffolding for the 3 distinct portals required for the ZK-Income Verification system.

## Structure
- **`bank-portal/`**: The Issuer/Oracle console.
- **`prover-wallet/`**: The User's wallet to generate proofs.
- **`verifier-service/`**: The public verification verification tool.

## Next Steps (for the Frontend Team)
1. Initialize each directory with `npm create vite@latest .`
2. Install dependencies: `npm install`
3. Link the local `wasm-pack` build from `../src`.
